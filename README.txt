# Guess The Esport Player (single-file)
Ouvrez `guess-esport-player.html` dans un navigateur, ou hébergez-le directement (Netlify Drop / GitHub Pages).

## Déploiement ultra-rapide
- **Netlify Drop**: glissez-déposez le fichier HTML sur https://app.netlify.com/drop
- **GitHub Pages**: créez un repo, uploadez le fichier, activez Pages (branch: main, folder: /root).
- **Vercel**: nouveau projet → importer repo contenant ce fichier → framework: Other → build: none.

Aucune étape de build, tout est dans un seul fichier (React UMD + Tailwind CDN + Babel standalone).
